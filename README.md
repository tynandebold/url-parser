# URL Parser

See what's returned in a parsed URL object.

## Setup

#### Install

```sh
$ npm install
```

## Run
```sh
$ npm start
```

## Deploy
```sh
$ npm run deploy
```
This command will generate a production build of the app and deploy it to GitHub Pages.
